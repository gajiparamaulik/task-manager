# Task Manager
Learning project demonstrating REST APIs, Express.js, MongoDB, authentication, and CRUD operations.

## 📁 Project Structure

| Folder/File | Description |
|-------------|-------------|
| `src/index.js`     | Starts the server |
| `src/app.js`       | Configures Express and wires up routes |
| `src/db/`          | Connects to MongoDB |
| `src/models/`      | Defines Mongoose schemas (data models) |
| `src/controllers/` | Contains business logic for each request |
| `src/routes/`      | Maps API endpoints to controllers |
| `src/middlewares/` | Middleware executed between request and controller |
| `src/validators/`  | Validates incoming request data |
| `src/utils/`       | Reusable helper functions |
